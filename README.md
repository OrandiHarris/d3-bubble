# &lt;d3-bubble&gt;
Bubble chart element for [Polymer](https://www.polymer-project.org) using [d3](http://d3js.org).

## Install

```
bower install d3-bubble
```

## Usage
### Tag

```
<d3-bubble data="[[data]]"></d3-bubble>
```

### Data Format

```
[
  {
    key: 'A',
    value: 11
  },
  {
    key: 'B',
    value: 15
  },
  {
    key: 'C',
    value: 6
  }
]
```
